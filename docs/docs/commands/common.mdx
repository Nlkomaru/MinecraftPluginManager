# コマンド

<CommandLine status="stable" command="mpm install <url>" />
urlからプラグインをインストールします。github, spigot, dev.bukkit, hanger, modrinthに対応しています。

<CommandLine status="stable" command="mpm uninstall <plugin>" />
プラグインをアンインストールします。

<CommandLine status="proposal" command="mpm list [--lock]" />
インストールされているプラグインの一覧を表示します。`--lock`をつけるとロックされているプラグインの一覧を表示します。

<CommandLine status="stable" command="mpm outdated <plugin>" />
プラグインの更新を確認します。

<CommandLine status="proposal" command="mpm outdatedAll" />
インストールされているプラグインの更新を確認します。

<CommandLine status="proposal" command="mpm update" />
/mpm outdated によって新しいバージョンがあるとされたプラグインを更新します。

<CommandLine status="stable" command="mpm help" />
ヘルプを表示します。

<CommandLine status="stable" command="mpm version" />
バージョンを表示します。

<CommandLine status="stable" command="mpm reload <plugin>" />
指定したプラグインをリロードします。

<CommandLine status="stable" command="mpm info <plugin>" />
プラグインの情報を表示します。詳細な情報（名前、バージョン、メインクラス、説明、作者、ウェブサイト、APIバージョンなど）が表示されます。

<CommandLine status="proposal" command="mpm search <plugin>" />
プラグインを検索します。

<CommandLine status="proposal" command="mpm lock <plugin>" />
プラグインをロックします。ロックをしたプラグインは、`mpm outdatedAll`で更新されません。

<CommandLine status="proposal" command="mpm unlock <plugin>" />
プラグインのロックを解除します。

<CommandLine status="proposal" command="mpm removeUnmanaged [--force]" />
mpm管理下にないプラグインを削除します。`--force`をつけると確認なしで削除します。

<CommandLine status="newly" command="mpm load <file>" />
指定したプラグインファイルを読み込みます。

<CommandLine status="beta" command="mpm unload <plugin>" />
指定したプラグインをアンロードします。

<CommandLine status="stable" command="mpm jarinfo <file>" />
指定したJARファイルのプラグイン情報を表示します。

## Usage  
まず初めに、`mpm install <url>`でプラグインをインストールします。  
インストールされたプラグインは`mpm list`で確認できます。  
個別のプラグインの更新を確認するには`mpm outdated <plugin>`を使用します。  
ですが、基本的には`mpm outdatedAll`で全てのプラグインの更新を確認することをおすすめします。  
また、`mpm outdated`で更新を確認した後は`mpm updateAll`で更新を行います。  
インストールされたプラグインをアンインストールするには`mpm uninstall <plugin>`を使用します。  

:::warning
    mpm管理下にないプラグインは削除されるため、/plugins/minecraftPluginManager/local にプラグインを移動してください。  
    この場合、`mpm updateAll`を実行時にプラグインがコピーされます。  
:::
